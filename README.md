Esta carpeta contiene:
- El codigo fuente de la aplicación PRESTLAN
- El script que genera la base de datos

PRESTLAN es un software realizado en ASP.NET que es necesario desplegar en un servidor web compatible (IIS)
Para que la aplicación funcione es necesario un servidor de base de datos compatible (SQL Server)
